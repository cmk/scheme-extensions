scheme-extensions

Extended recursion scheme library built on data-fix's MuFixNu types. Provides paramorphisms, zygomorphisms, Elgot algebras, mutual recursion, pre/postpromorphisms, generic streaming metamorphisms (Gibbons), Cons pattern functor, and Kan extension connections (Day-based structural comparison, Codensity/StateT isomorphisms, Yoneda hoist fusion).
